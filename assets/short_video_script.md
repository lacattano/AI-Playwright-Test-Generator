# Short Video Script — AI Playwright Test Generator

Duration: 45 seconds (30–60s range)
Format: Screen capture + captions + short voiceover

00–05s (Hook)
- On-screen text: "From user story to working Playwright test in under a minute"
- Visual: Split-screen: Streamlit UI (left) and CLI output (right)
- Caption: "No coding required — GUI or CLI"

05–15s (Set context)
- Show: Enter base URL and paste short user story (highlight acceptance criteria) in Streamlit; show equivalent CLI command on right
- Caption: "Paste a user story"

15–30s (Generate & inspect)
- Click "Generate Test"; show spinner (time-lapse if slow) and CLI generation logs
- Reveal generated Python code; zoom to selectors (get_by_* calls)
- Caption: "AI scrapes real DOM selectors — no hallucination"

30–34s (Run)
- Click "Run Now" or show CLI pytest invocation; show brief browser execution or console test run
- Caption: "Execute in a real browser"

34–36s (Results)
- Show passing result summary: "2 tests passed" in both UI and CLI
- Caption: "Fast, actionable results"

36–40s (Reports)
- Open HTML report preview; show screenshots and coverage / JSON payload briefly
- Caption: "Shareable reports (HTML / JSON)"

40–45s (Export & CTA)
- Click export (Python / JSON / HTML); overlay CTA: "Try it: github.com/lacattano/AI-Playwright-Test-Generator"

Recording tips:
- Speed up the generate step 2x to keep video snappy
- Add captions for clarity; keep voiceover concise
- Keep total length ≤60s for sharing via phone
